# Онлайн-магазин

## Настройка сборки

```bash
#Установите нужные библиотеки
$ cd shops
$ pip install -r ./requirements.txt
$ python manage.py runserver

```
## Для работы будет необходимо скачать Redis:
Качать тут:[Redis](https://redis.io/)

```bash
$ cd redis-(Номер версии)
$ src/redis-server

```
